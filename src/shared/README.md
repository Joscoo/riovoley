# Shared

Utilidades y componentes transversales para features migradas.

- `ui/`: componentes de UI reutilizables basados en Tailwind
  - `ui/layout/`: layout base de paneles por rol (`RolePanelLayout`, `RoleSidebar`)
  - `ui/feedback/`: feedback global (`Toast`)
- `lib/`: helpers puros y tipos de resultado
- `config/`: configuracion runtime compartida
- `infrastructure/`: gateways compartidos (auth/communications)
- `domain/`: servicios y reglas de dominio reutilizables
